# 🤝 KeenKeeper

KeenKeeper is a modern friend interaction dashboard designed to help you nurture and track your most meaningful relationships. It provides a visual overview of your social connections, interaction history, and analytics to ensure you never lose touch with the people who matter most.

---

## 🚀 Technologies Used

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Data Visualization:** Recharts
- **State Management:** React Context API
- **Icons:** React Icons
- **Data Source:** Local JSON

---

## ✨ Key Features

### 1. Friend Analytics Dashboard

Gain insights into your social habits. The dashboard visualizes interaction statistics by type (Calls, Texts, Videos) using interactive charts, helping you see where you're investing your time.

### 2. Personal Relationship Manager

Dedicated dynamic routes (`/friends/[id]`) for every friend. View personalized interaction histories, status updates (On Track vs. Overdue), and specific action logs for each individual.

### 3. Activity Timeline & Stats

A centralized timeline view that logs every interaction chronologically. Filter through your history to see past calls, messages, or video chats, all wrapped in a fully responsive, mobile-friendly interface.
